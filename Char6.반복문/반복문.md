### 1. While문

* While문을 사용한 반복문

  ```c
  #include <stdio.h>
  
  int main(void)
  {
  	int a = 1;
  
  	while (a < 10)
  	{
  		a = a * 2;
  	}
  	printf("a : %d\n", a);
  
  	return 0;
  }
  ```







---

### 2. for 문

* for 문을 사용한 반복문

  ```c
  #include <stdio.h>
  
  int main(void)
  {
  	int a = 1;
  	int i;
  
  	for (i = 0; i < 3; i++)
  	{
  		a = a * 2;
  	}
  	printf("a : %d\n", a);
  
  	return 0;
  }
  ```



---

#### 3. do ~ while 문

* do while문을 사용한 반복문

  ```c
  #include <stdio.h>
  
  int main(void)
  {
  	int a = 1;
  
  	do
  	{
  		a = a * 2;
  	} while (a < 10);
  	
  	printf("a : %d\n", a);
  	
  	return 0;
  }
  ```



---

### 4. 중첩 반복문

* 중첩반복문을 사용한 별 출력

  ```c
  #include <stdio.h>
  
  int main(void)
  {
  	int i, j;
  
  	for (i = 0; i < 3; i++)
  	{
  		for (j = 0; j < 5; j++)
  		{
  			printf("*");
  		}
  		printf("\n");
  	}
  	
  	return 0;
  }
  ```
